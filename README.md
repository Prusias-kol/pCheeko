# pCheeko
CHEEKO

To install, run  
`git checkout https://github.com/Prusias-kol/pCheeko main`

## How to Use
To use the script,  type 
`cheeko <x>`
where x is the number of advs to spend farming bumcheeks. 
`cheeko *`
will spend all your adventures.
